# 医院故障工单管理系统

适用于医院信息科的 IT 工单管理 Web 系统。

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 📊 **仪表盘** | 本月工单统计、故障类型分布、人员排行、近7日趋势 |
| 📋 **工单列表** | 搜索/筛选/分页，支持按关键词、类型、人员、日期筛选 |
| ✏️ **新建工单** | 手动创建单个工单 |
| 🪄 **批量生成** | 自动随机生成测试/模拟工单（软件/硬件/打印机/协助） |
| 📁 **Excel导出** | 导出当前筛选条件下的工单为 Excel |
| 👥 **人员管理** | 增删改维护人员名单 |
| 📖 **方案模板** | 150条维修方案模板，支持编辑和重置 |
| 🔐 **登录认证** | 简单账号密码保护 |

## 🚀 快速启动

### Windows 用户

1. 安装 [Python 3.8+](https://www.python.org/downloads/)（安装时勾选 "Add Python to PATH"）
2. 双击 `start.bat`，等待自动安装依赖和启动
3. 浏览器打开 `http://127.0.0.1:5000`

### Linux / macOS 用户

```bash
chmod +x deploy.sh
./deploy.sh
```

## 🔑 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |

登录后请在 **数据管理 > 人员管理** 中维护人员名单。

## 📁 目录结构

```
hospital-workorder/
├── app.py              # 主入口（双击启动）
├── config.py           # 配置 + 方案模板
├── models.py           # 数据库模型
├── routes/             # Flask 路由
│   ├── auth.py         # 登录退出
│   ├── main.py         # 仪表盘
│   ├── orders.py       # 工单管理
│   └── data.py         # 数据管理
├── services/           # 业务逻辑
│   ├── address.py      # 全院地址数据
│   ├── generator.py    # 工单生成引擎
│   └── matcher.py      # 方案匹配引擎
├── templates/          # 前端模板
├── requirements.txt    # Python 依赖
├── deploy.sh           # Linux/Mac 部署脚本
└── start.bat           # Windows 快速启动
```

## ⚙️ 自定义

- **人员名单**: 登录后在 数据管理 > 人员管理 中添加
- **方案模板**: 登录后在 数据管理 > 方案模板 中编辑
- **方案模板重置**: 可一键恢复为出厂默认值
- **默认端口**: 如需修改，在 `app.py` 最后改 `port=5000`
