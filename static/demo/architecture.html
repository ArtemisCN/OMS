医院IT工单管理系统 · 系统架构

医院IT工单管理系统 — PC / Mobile / 小程序 / Android 多端统一架构

多端前端 · 统一体验

  • • PC Web端 — Flask + Jinja2 模板渲染，后台管理界面
  • • Mobile Web端 — 响应式设计，适配手机与平板浏览器
  • • 微信小程序 — 原生小程序框架，微信生态内无缝使用
  • • Android端 — Java / Kotlin 原生 App，独立运行

后端基础设施 · Python 驱动

  • • Nginx — HTTPS 终止、静态资源服务、请求转发
  • • Gunicorn — Python WSGI HTTP 服务器，多 worker 并发
  • • Flask — 工单 CRUD、用户认证、权限管理、API 路由
  • • SQLite — 零配置文件数据库，工单/用户/配置持久化

外部集成 · 消息推送

  • • 企业微信通知 — 工单创建/流转/完成时推送至企业微信群
  • • 微信订阅消息 — 主动向用户微信发送工单状态更新
  • • REST API — Flask 提供统一接口供所有端调用
  • • SQLAlchemy ORM — 数据库抽象层，便于迁移扩展
